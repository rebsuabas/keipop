== Description ==
Musican is a top notch music WordPress theme, suitable for musicians, artists, music bands, producers or anyone working in the music industry. The theme design is creative and follows the latest design trends. Checkout demo: https://demos.filathemes.com/musican/


== Frequently Asked Questions ==

= Does this theme support any plugins? =

Musican includes support for Infinite Scroll in Jetpack, Elementor & The Event Calendar.
